**Lua-enhanced verbatim / Luaにより拡張されたverb**

---

## Overview / 概要
`KKluaverb` is a LaTeX package which provides a Lua-enhanced `\verb` command, `\KKverb`. It can be used in the table of contents, index, underline commands (e.g., `\underLineKK` provided by the luwa-ul package), `tblr` environment and so on.<br>
`KKluaverb`は、Luaによって拡張された`\verb`である`\KKverb`を提供します。目次や索引、下線コマンド（例えばluwa-ulパッケージの提供する`\underLineKK`）、及び`tblr`環境などのなかでも使用可能です。
